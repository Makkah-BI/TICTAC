# Tic Tac Toe

A classic Tic Tac Toe game for two players. Built with HTML, CSS, and JavaScript.

## Features
- 3×3 game grid with interactive tiles
- Automatic win detection (rows, columns, diagonals)
- Draw detection
- Game status display
- Reset button to start a new game

## Technologies
- HTML5
- CSS3 (Flexbox/Grid)
- Vanilla JavaScript
